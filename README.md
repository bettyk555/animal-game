# animal-game
Игра "Угадай животное" — компьютер задаёт вопросы "да/нет" и угадывает животное.  Если ошибся — обучается: добавляет новое животное и вопрос. Функционал: бинарное дерево решений, рекурсивный обход, стек отмены,  статистика эффективности вопросов с сортировкой, префиксные суммы,  сохранение/загрузка в JSON. Проект 2 семестр, командная работа.
