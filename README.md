# Игра "Угадай животное"

## Описание
Консольная игра, в которой компьютер пытается угадать загаданное животное, задавая вопросы с ответами "да"/"нет". Если компьютер не угадывает, он обучается на своих ошибках.
Работу выполнили Пак Валерия и Пшеничникова Кристина.

## Функционал
- Бинарное дерево решений для хранения вопросов и животных
- Рекурсивный обход дерева
- Обучение: добавление новых животных и вопросов
- Стек для отмены последних ответов
- Статистика эффективности вопросов
- Префиксные суммы для анализа глубины угадывания

## Как запустить
```bash
python main.py
```
